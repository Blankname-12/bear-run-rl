# 会话 3 概述

> 日期: 2026-05-18
> 主题: 截屏方案调试 + 奖励重构 + 仿真环境开发

## 本会话完成的工作

### 1. 截屏方案最终选择
- 测试了 scrcpy/PyAV/screenrecord/ADB raw/minicap/window 六种方案
- 结论: **window (mss) 是唯一可用方案**（17ms/帧, 58FPS）
- 代价: 模拟器窗口必须保持可见

### 2. 奖励函数重构
- 移除帧差法障碍物奖励（噪声过大）
- 保留四个可靠信号: 存活/死亡/红心/边界
- 暖机从固定时间改为红心检测

### 3. 仿真环境（核心产物）
- 纯 Python 伪 3D 跑酷游戏
- MLP(向量) + CNN(像素) 双模式 PPO 训练
- 可视化 AI 试玩

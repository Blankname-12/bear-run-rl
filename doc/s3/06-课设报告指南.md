# 强化学习课设报告撰写指南

### 第一章: 引言 (1-2页)
- 强化学习在游戏AI中的应用背景
- 跑酷游戏(Temple Run类)的RL挑战

### 第二章: 相关工作 (1-2页)
- DQN/PPO算法简介

### 第三章: 真机方案探索 (2-3页, 作为失败案例分析)

- 3.2 截屏方案对比 (四种方案性能表)

- 4.4 奖励设计 (四轮迭代)
- 4.5 碰撞判定

### 第五章: 算法与模型 (2-3页)
- 5.1 PPO算法原理 (clip机制/GAE)
- 5.2 MLP架构 (完美信息上界)
- 5.3 CNN架构 (从像素学习)
- 5.4 超参选择

### 第六章: 实验与分析 (3-4页)
- 6.1 MLP训练 (向量模式)
- 6.2 CNN训练 (像素模式)
- 6.3 对比分析 (收敛速度/最终得分/学习曲线)
- 6.4 消融实验 (可选)

### 第七章: 结论与展望 (1页)
- 工作总结
- 真机vs仿真对比讨论
- 未来方向

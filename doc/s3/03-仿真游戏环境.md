# 仿真跑酷游戏环境

> 文件: `simple_game.py`, 新建于会话3

## 设计理念
- 纯 Python + pygame, 零外部依赖(除pygame)
- 完美状态访问: 障碍物位置/类型/赛道/分数全部精确
- 双观测模式: 向量14维(MLP用) / 像素160×160(CNN用)
- 伪3D透视渲染, 模仿Temple Run视觉风格

## 游戏参数
```python
N_LANES = 3          # 左/中/右
N_ACTIONS = 5        # 跳/滑/左移/右移/直跑
VIEW_W = 480         # 渲染宽度
VIEW_H = 640         # 渲染高度
speed_init = 2.5     # 初始速度
speed_max = 10.0     # 最大速度
speed_accel = 0.0005 # 每步加速
```

## 观测空间

### 向量模式 (14维)
```
[0-2]:   当前赛道 one-hot (左/中/右)
[3-5]:   近距离障碍物所在赛道 one-hot
[6]:     近距离障碍物需要跳跃? (跨栏)
[7]:     近距离障碍物需要滑铲? (矮栏)
[8-10]:  中距离障碍物所在赛道 one-hot
[11]:    中距离需要跳跃?
[12]:    中距离需要滑铲?
[13]:    速度归一化
```

### 像素模式
- (160, 160, 3) RGB np.float32 [0,1]
- 直接从 pygame Surface 抓取: `pygame.surfarray.array3d(small)`
- 零截屏开销, 不需要 minicap/scrcpy/ADB

## 碰撞判定
```python
if obstacle_lane == player_lane and 480 < obstacle_wy < 570:
    if otype==0(跨栏) and action!=0(跳): hit
    if otype==1(矮栏) and action!=1(滑): hit
    if otype==2(左路障) and lane==0:     hit
    if otype==3(右路障) and lane==2:     hit
```

## 障碍物生成
- 4种类型: 跨栏(跳)、矮栏(滑)、左路障(右移避开)、右路障(左移避开)
- 最小间距: 160 单位
- 生成概率: 30%/步
- 从远处(900单位)滚入, 超过玩家(-80)消失

## 渲染特性（会话3多次迭代优化）
1. 天空渐变 (深蓝→浅蓝)
2. 远山轮廓
3. 绿色草地 (填充多边形)
4. 灰色路面 (填充多边形, 透视梯形)
5. 2条粗白实线车道分隔 (汇聚消失点)
6. 3条虚线车道中心标记 (汇聚消失点)
7. 红色护栏 (路边缘)
8. 路边树木 (远近不同大小, 深度感)
9. 大气雾化 (`_fog_blend`, 远处颜色褪向天蓝色)
10. 速度线 (高速时白色短线)
11. 玩家角色: 影子椭圆 + 腿 + 身体矩形(高光) + 头圆 + 头发椭圆
12. 障碍物3D效果: 跨栏红杆+支柱, 矮栏危险条纹, 路障半矩形
13. 金币: 外圈金色+内圈亮黄光晕
14. HUD: 半透明黑底面板, Score/Speed/Lane
15. Game Over: 半透明黑遮罩+红色大字

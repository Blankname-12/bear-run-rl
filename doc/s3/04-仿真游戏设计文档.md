# 仿真跑酷游戏 — 完整设计文档

## 设计目标

替代 Android 模拟器方案，提供:
1. 完美状态访问（不需要截屏/视觉检测）
2. 高速训练（毫秒级/局）
3. 可控实验条件（可复现、可调参）
4. 可选视觉渲染（CNN 从像素学习）

## 整体架构

```
SimpleRunner (Gym 兼容接口)
├── 游戏逻辑 (纯 Python)
│   ├── 3 赛道 × 4 障碍物类型 × 金币
│   ├── 碰撞判定 (精确 lane + type + y 坐标)
│   ├── 速度渐进递增 (课程学习)
│   └── 障碍物生成 (概率 + 间距控制)
├── 观测输出
│   ├── mode="vector": 14 维 np.float32
│   └── mode="visual": (160,160,3) np.float32 [0,1]
├── 渲染 (pygame)
│   ├── Surface 480×640 伪 3D 透视
│   └── 多层绘制 (天空→远山→地面→车道→树→障碍→硬币→玩家→HUD)
└── 奖励计算
    ├── 分数增量 × 0.1
    ├── 存活 +0.01/步
    ├── 闪避 +0.5
    ├── 金币 +1.0
    ├── 撞墙 -0.3
    └── 死亡 -5.0
```

## 障碍物系统

### 四种类型
```python
OBSTACLE_JUMP = 0   # 跨栏: 必须跳跃闪避
OBSTACLE_SLIDE = 1  # 矮栏: 必须滑铲闪避
OBSTACLE_LEFT = 2   # 左路障: 处于左车道时碰撞,需右移避开
OBSTACLE_RIGHT = 3  # 右路障: 处于右车道时碰撞,需左移避开
```

### 生成逻辑
```python
# 每步调用:
max_wy = max([o[2] for o in self.obstacles], default=self._last_spawn_y)
if max_wy < 900:  # 远端有空间
    if random() < 0.30:
        lane = randint(0, 2)
        otype = randint(0, 3)
        spawn_y = max_wy + 160 + randint(0, 80)
        self.obstacles.append((lane, otype, spawn_y))
```

### 生命周期
```
生成(wy≈900) → 向下滚动(wy -= speed) → 碰撞(wy≈500-570) → 离开(wy<-80)
                                                                    ↓
                                                           闪避成功 → reward+0.5
```

### 碰撞判定
```python
for lane, otype, wy in self.obstacles:
    if lane != self.lane:
        continue  # 不同赛道, 不影响
    if 480 < wy < 570:  # 在碰撞区域内
        if otype==0 and action!=0:   # 跨栏没跳
            hit = True
        elif otype==1 and action!=1: # 矮栏没滑
            hit = True
        elif otype==2 and lane==0:   # 左路障且在最左
            hit = True
        elif otype==3 and lane==2:   # 右路障且在最右
            hit = True
```

## 观测空间

### 向量模式 (14维, `_vector_obs()`)
```python
obs = np.zeros(14, dtype=np.float32)

# 赛道 one-hot [0-2]
obs[self.lane] = 1.0

# 找最近障碍物 (近: wy 420-600, 中: wy 600-820)
near = max([o for o in obstacles if 420 < o[2] < 600], key=lambda o: o[2])
mid  = max([o for o in obstacles if 600 < o[2] < 820], key=lambda o: o[2])

# 近距离 [3-7]
if near:
    lane, otype, wy = near
    obs[3+lane] = 1.0         # 所在赛道
    obs[6] = 1.0 if otype==0 else 0.0  # need_jump
    obs[7] = 1.0 if otype==1 else 0.0  # need_slide

# 中距离 [8-12]
if mid:
    obs[8+lane] = 1.0
    obs[11] = 1.0 if otype==0 else 0.0
    obs[12] = 1.0 if otype==1 else 0.0

# 速度 [13]
obs[13] = (speed - 2.5) / (10.0 - 2.5)
```

### 像素模式 (`_observe()` visual 分支)
```python
if self.mode == "visual":
    self.render()                            # 画到 self._screen (Surface 480×640)
    small = pygame.transform.smoothscale(    # 缩放到 160×160
        self._screen, (160, 160))
    arr = pygame.surfarray.array3d(small)    # → (160, 160, 3) uint8
    return arr.transpose(1, 0, 2) / 255.0   # → (H, W, 3) float32 [0,1]
```

## 伪 3D 透视投影

### 核心公式 `_project(lane_idx, world_y)`
```python
t = max(0.0, min(1.0, world_y / 1000.0))   # 世界坐标→[0,1]归一化
vanish_x = 240                              # 消失点 X
lane_x = [80, 240, 400][lane_idx]           # 赛道在画面底部的 X

# 透视: 越远越靠近消失点
x = vanish_x + (lane_x - vanish_x) * (1 - t)
y = 550 + (180 - 550) * t                   # 越远越靠近消失点 Y
scale = 1.0 - t * 0.85                      # 近大远小
```

### 大气雾化 `_fog_blend(rgb, t)`
```python
# t 越大(越远), 颜色越偏向天蓝色
fog = (120, 180, 230)
f = min(1.0, t * 0.7)
r = int(rgb[0] * (1-f) + fog[0] * f)
g = int(rgb[1] * (1-f) + fog[1] * f)
b = int(rgb[2] * (1-f) + fog[2] * f)
```

for i in range(640):

### 2. 远山
4 个不同高度和位置的三角形，用 Earth-tone 颜色。

```python
road_w_bottom = 400                          # 路底宽
road_l_bottom = 240 - 200                    # 路底左边界
road_r_bottom = 240 + 200                    # 路底右边界

# 左草地梯形
pygame.draw.polygon(surf, (80,140,60),

# 右草地梯形 (对称)
# 路面梯形
pygame.draw.polygon(surf, (130,125,115),
    [(road_l_bottom,640), (road_l_top,100), (road_r_top,100), (road_r_bottom,640)])
```

### 4. 车道分隔线
- 2 条白色粗实线从消失点发散（lane_frac=1/3, 2/3）
- 3 条灰色虚线（lane_frac=1/6, 3/6, 5/6），从近到远绘制

### 5. 红色护栏
车道边缘两条红线（lane_frac=0.05, 0.95）

### 6. 路边树木
左右各 4 棵树 (world_y=200, 400, 600, 850)，树干棕色矩形 + 树冠绿色椭圆。

### 7. 障碍物 `_draw_obstacle_3d(lane, otype, wy)`
每种障碍物有独特的 3D 外观:
- **跨栏(red)**: 红色横杆 + 两个棕色支柱
- **矮栏(orange)**: 贴地橙色块 + 黑色危险条纹
- **路障(purple)**: 半个矩形（左路障在左边，右路障在右边）

所有障碍物应用 `_fog_blend` 实现远距离雾化。

### 8. 金币 `_draw_coin_3d(lane, wy)`
外圈深金 + 内圈亮黄光晕。

### 9. 玩家 `_draw_player(px, s)`
- 地面椭圆阴影（半透明黑色）
- 两条腿（深蓝线段）
- 身体矩形（蓝色 + 浅蓝高光左半）
- 头圆形（肤色）
- 头发椭圆（深棕）

### 10. HUD
半透明黑底面板 + Score/Speed/Lane 三行文字。

### 11. 速度线
speed > 4 时随机位置的白线（alpha 随 speed 增长）

### 12. Game Over
半透明黑遮罩 + 红色大字 "GAME OVER"

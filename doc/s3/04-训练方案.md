# 训练方案

> 文件: `train_simple.py`, 新建于会话3

## 网络架构

### MLP (完美信息上界)
```
输入: 14维向量
MLP: Linear(14→128)→ReLU→Linear(128→128)→ReLU
Actor: Linear(128→5)
Critic: Linear(128→1)
参数: ~5K
```

### CNN (从像素学习)
```
输入: (160,160,3) RGB
Conv1: 32ch, k8, s4 → 39×39
Conv2: 64ch, k4, s2 → 18×18
Conv3: 64ch, k3, s1 → 16×16
Flatten: 64×16×16 = 16384d
FC: Linear(16384→256)→ReLU
Actor: Linear(256→5)
Critic: Linear(256→1)
```

## PPO 超参
```python
lr = 1e-3          # 仿真环境下可更高
gamma = 0.99       # 折扣因子
lambda = 0.95      # GAE参数
clip = 0.2         # PPO clip范围
epochs = 4         # 每批数据重复使用次数
batch = 128        # mini-batch大小
ent_coef = 0.02    # 熵正则化系数
critic_coef = 0.5  # Critic损失权重
max_grad_norm = 1.0 # 梯度裁剪
```

## 训练命令
```bash
# MLP 向量模式 (~1分钟)
python train_simple.py

# CNN 像素模式 (~10分钟)
python train_simple.py --visual

# 对比实验 (先后跑两种)
python train_simple.py --compare

# 边训练边看画面
python train_simple.py --render

# 评估训练好的模型
python train_simple.py --eval models/simple_cnn_best.pth
```

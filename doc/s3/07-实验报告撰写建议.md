## 建议章节结构（7章）

### 第一章: 引言（1-2页）

**可写内容**:
- 强化学习在游戏 AI 领域的应用背景（简单介绍 DQN 玩 Atari、AlphaStar 打星际等经典案例）
- Temple Run 类跑酷游戏的 RL 挑战：需要视觉感知 + 快速反应 + 长期规划
- 本研究的目标：训练一个智能体在跑酷游戏中自主决策，能识别并闪避障碍物
- 研究意义：探索 RL 在移动游戏自动化中的应用前景

**关键引用**:
- 真机方案面临的技术挑战（I/O 瓶颈、视觉噪声、广告干扰）
- 从真机到仿真的方案转变动机

## 可引用的具体数据

### 真机方案
- exp18: 462局, 步数均值101, reward均值-32.09, 无学习趋势
- exp22: 276局, 步数均值92, reward均值-31.06, PPO更新99次
- exp40: 855局, 步数均值19, reward均值-18.64, 99.9%暴毙
- ADB raw: 651ms(最快)~1361ms(最慢), 平均1001ms, 1FPS
- window(mss): 17ms/帧, 58FPS
- minicap: 50ms/帧, 10-20FPS

### 仿真方案
- MLP: 14维→128→128→5, ~19K参数
- CNN: 160×160×3→5, ~500K参数
- PPO超参: lr=1e-3, γ=0.99, λ=0.95, clip=0.2
- 训练速度: MLP ~1分/2000局, CNN ~10分/2000局
- 最终性能: 运行 `python train_simple.py` 获取

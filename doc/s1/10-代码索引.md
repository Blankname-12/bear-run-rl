# 10 — 代码索引

每个文件的关键函数/类及行号参考。

## config.py
- `BASE_DIR` — 项目根目录
- `CAPTURE_MODE` — 截屏模式（minicap/window/adb）
- `ACTION_SPACE / ACTION_NAMES` — 动作空间
- `HEART_*` — 红心检测参数
- `PPO 超参` — LEARNING_RATE, GAMMA, GAE_LAMBDA, CLIP_EPSILON 等
- `setup_run_dir()` — 创建 runs/train/expN/

## game_env.py
### 底层截屏
- `_adb_screencap()` — ADB 截屏（兜底方案）
- `get_screen_numpy()` — 统一截屏入口（优先 fast_io）
- `_crop_game_area(frame)` — 裁剪游戏区域（坐标乘 _capture_scale）
- `preprocess_frame(frame)` — 裁剪+resize+归一化（优先OpenCV）

### 红心检测
- `_detect_heart_pixels(rgb_frame)` — 红心区域裁剪+像素统计
- `_count_red_pixels(rgb_patch)` — HSV 红色计数（OpenCV）
- `_heart_lost(raw, fast)` — 心跳丢失检测（3帧投票/单帧快速）
- `_verify_death()` — 二次验证（点取消→查5帧→区分广告/死亡）

### GameEnv 类
- `__init__()` — 初始化
- `reset()` — 重置环境（清计数、设warmup、init帧栈）
- `step(action_idx)` — 训练步（先检测→再执行→奖励）
- `observe(save_ss)` — 观察步（截屏+检测，采集用）
- `send_action(action_idx)` — 快速发滑动（采集用）
- `restart_game()` — 死亡重启（点重新开始→点开始跑酷）
- `get_state()` — 拼装 15 通道状态（4帧+3赛道）
- `_init_frame_stack()` / `_push_frame()` — 帧堆栈管理
- `_check_danger(raw)` — 障碍逼近检测（帧差法）
- `_manage_screenshots(raw)` — 环形截图管理
- `_tap(x,y)` — 同步点击

### 触控辅助
- `_get_fast_touch()` — 获取 FastTouch 单例
- `_swipe_via_adb(x1,y1,x2,y2)` — ADB 回退滑动
- `_tap_via_adb(x,y)` — ADB 回退点击
- `action_swipe(action_idx)` — 训练模式滑动入口

## fast_io.py
### FastTouch 类
- `swipe(x1,y1,x2,y2,dur)` — Popen fire-and-forget 滑动
- `tap(x,y)` — Popen fire-and-forget 点击
- `tap_sync(x,y)` — subprocess.run 同步点击（重启用）

### MinicapCapture 类
- `connect()` — 启动服务端+连接socket
- `get_frame()` — drain旧帧+返回最新帧JPEG→numpy
- `close()` — 断开socket+清理转发
- `_start_server()` — 推送二进制+启动minicap
- `_push_and_setup()` — push+安装so到system/lib64

### 全局函数
- `get_touch()` — FastTouch 单例
- `get_capture()` — MinicapCapture 单例
- `capture_frame()` — 统一截屏入口（minicap→DXGI→ADB）
- `shutdown_io()` — 清理资源

## agent.py
### PPONetwork
- `forward(x)` → (logits, value)
- `get_action(state, deterministic)` → (action, log_prob, entropy, value)
- `evaluate(states, actions)` → (log_probs, entropy, values)

### RolloutBuffer
- `push(s,a,log_prob,r,v,done)`
- `get_all()` → 6个numpy数组

### PPOAgent
- `select_action(state, eval_mode)` → 动作选择
- `store_transition(...)` → 存经验
- `update()` → PPO 更新（512步触发）
- `save_checkpoint()` / `load_checkpoint()` / `save_best_model()` / `save_temp_model()`
- `_compute_gae(...)` → GAE 优势计算

## collect_data.py
- `collect(resume)` — 主采集循环（v5极简同步版）
- `key_to_action(key)` — 键盘→动作映射
- `read_keyboard()` — msvcrt 读键
- `save_data(session_dir, ...)` — 保存 states.npy + actions.npy + meta.txt
- `find_latest_session()` / `load_existing_data()` — 会话管理

## pretrain.py
- `pretrain()` — 行为克隆主函数
- `find_data_paths()` — 自动找最新session

## train.py
- `train(resume)` — PPO训练主循环
- `evaluate()` — 推理模式
- `TrainingLogger` — CSV日志记录器

## setup_minicap.py
- `get_device_info()` — 检测ABI+SDK
- `download_and_extract()` — 引导下载
- `push_to_device()` — 推送二进制
- `test_minicap()` — 验证运行

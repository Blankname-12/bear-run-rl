# 09 — 环境搭建

## 1. 软件准备

- 雷电模拟器（LDPlayer）— Android 9.0 版本
- ADB 工具（Android SDK Platform Tools）
- Conda（Anaconda/Miniconda）
- Python 3.x + PyTorch + CUDA

## 2. Python 环境

```bash
conda create -n pytorch python=3.11
conda activate pytorch
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
pip install numpy opencv-python pillow matplotlib
pip install mss pywin32   # Windows DXGI 捕获（可选）
```

## 3. 模拟器配置

1. 安装雷电模拟器，创建 Android 9.0 实例
2. 设置 → 其他设置 → 开启 ADB 调试
3. 分辨率设为 1080×1920（手机竖屏模式）
4. 安装"熊大快跑"游戏 APK

## 4. ADB 连接

```bash
adb connect 127.0.0.1:5555
adb devices   # 确认 device 状态
```

雷电模拟器默认 ADB 端口为 5555。

## 5. minicap 安装（需 Root）

### 5.1 下载二进制

从 https://github.com/DeviceFarmer/minicap/releases 下载 `minicap-2.7.3.zip`，解压到 `bin/minicap/`。

### 5.2 组织目录结构

```
bin/minicap/
  x86_64/minicap                        ← 来自 libs/x86_64/minicap
  android-28/x86_64/minicap.so          ← 来自 jni/.../aosp/libs/android-28/x86_64/minicap.so
```

### 5.3 推送并安装

```bash
adb push bin/minicap/x86_64/minicap /data/local/tmp/minicap
adb push bin/minicap/android-28/x86_64/minicap.so /data/local/tmp/minicap.so
adb shell chmod 755 /data/local/tmp/minicap

# 安装 so 到系统库（需要 root）
adb shell su -c 'mount -o rw,remount /'
adb shell su -c 'cp /data/local/tmp/minicap.so /system/lib64/minicap.so'
adb shell su -c 'chmod 644 /system/lib64/minicap.so'
```

### 5.4 验证

```bash
adb shell /data/local/tmp/minicap -P 1080x1920@1080x1920/0 -s > test.jpg
# 应该生成一张 1080×1920 的 JPEG 截图
```

### 5.5 运行 setup 脚本

```bash
python setup_minicap.py
# 自动检测 ABI/SDK，推送，测试
```

## 6. 验证整个系统

```bash
# 1. 确认 ADB 连接
adb devices

# 2. 确认 minicap 能启动
adb shell "nohup /data/local/tmp/minicap -P 1080x1920@1080x1920/0 -r 15 -S >/dev/null 2>&1 &"
adb shell pidof minicap   # 应有 PID

# 3. 测试截屏
python -c "from fast_io import capture_frame; f=capture_frame(); print(f.shape)"
# 应输出 (1920, 1080, 3)

# 4. 测试触控
python -c "from fast_io import FastTouch; ft=FastTouch('127.0.0.1:5555'); ft.tap(540,960)"
# 模拟器应有点击反应

# 5. 启动数据采集
python collect_data.py
```

## 7. 常用命令

```bash
python collect_data.py              # 采集数据
python collect_data.py --resume     # 续接
python pretrain.py                  # 预训练
python pretrain.py --epochs 50      # 指定轮数
python train.py                     # PPO 训练
python train.py --new               # 从头训练
python train.py --eval              # 推理评估
```
